# 技能库

本目录包含可复用的 AI 技能提示词。这些技能是方法选择器，用于指导 AI Agent 执行特定类型的任务。

## 使用说明

- 技能提示词必须与 owner docs 结合使用，不能替代 owner docs
- 每个技能定义了执行某类任务的方法论和检查清单
- 使用前应根据项目实际情况进行定制

## 可用技能

| 技能 | 用途 |
|---|---|
| [writing-conventions.md](writing-conventions.md) | 文档编写规范（中英文间距、PlantUML、Markdown） |
| [document-audit-prompt.md](document-audit-prompt.md) | 文档审计：挑战需求/设计/架构文档 |
| [plan-audit-prompt.md](plan-audit-prompt.md) | 计划审计：在实施前挑战计划 |
| [closure-audit-prompt.md](closure-audit-prompt.md) | 闭合审计：独立的完成度审查 |
| [multi-dimensional-audit-prompt.md](multi-dimensional-audit-prompt.md) | 多维审计：跨维度的风险挑战 |
| [open-ended-audit-prompt.md](open-ended-audit-prompt.md) | 开放式审计：发现隐藏风险 |
| [requirement-gap-retrospective-prompt.md](requirement-gap-retrospective-prompt.md) | 需求差距回顾：诊断需求偏差原因 |
