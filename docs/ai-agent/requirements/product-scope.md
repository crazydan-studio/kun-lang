# 产品范围

## Kun 语言定位

Kun（鲲）是一款面向 Linux 的函数式脚本语言，定位于替代 Shell 脚本用于系统管理自动化和命令行工具编排。

## 目标用户

- Linux 系统管理员
- DevOps 工程师
- 需要编写自动化脚本的开发者
- 对函数式编程感兴趣的高级用户

## 核心价值

1. **类型安全**：消除 Shell 脚本中因缺乏类型检查导致的运行时错误
2. **命令组合**：将 Linux 命令组合从字符串拼接升级为类型驱动的函数复合
3. **安全沙箱**：默认最小权限，从根本上降低脚本的安全风险
4. **表达式导向**：简洁一致的语法，提升脚本的可读性和可维护性

## 不包含

- GUI 应用开发
- 系统级编程（替代 C/Rust）
- 跨平台支持（仅 Linux）
- Web 开发

## 与其他语言的关系

| 语言 | 关系 |
|---|---|
| Shell (bash/sh) | Kun 的替代目标 |
| Python | 在自动化脚本领域的竞争者 |
| Haskell/Elm | 语法和类型系统的灵感来源 |
| Rust | 宿主语言 Zig 的类似定位 |
